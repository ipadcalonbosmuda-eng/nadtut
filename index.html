FAQ

What is Nadz Tools?

Nadz Tools is a comprehensive Web App for deploying tokens, locking tokens & liquidity, managing vesting schedules, and multi-sending on Monad Network. It provides a simple and transparent platform for all your token management needs.

Where do I deploy and manage tokens?

Inside the Nadz Tools Web App at app.nadz.tools. This website is for information only - all actual token operations happen in the Web App.

Which networks are supported?

Nadz Tools is exclusive to Monad Network. We focus solely on Monad to provide the best experience and security. Cross-chain support is not planned.

How secure is Nadz Tools?

Nadz Tools prioritizes security first with on-chain verification, automatic contract verification, and transparent proof systems. All operations are verifiable on the Monad blockchain.

What fees are involved?

Nadz Tools charges minimal fees for operations, with transparent pricing displayed in the Web App. You only pay for the actual blockchain transaction costs plus a small service fee.

Do I need technical knowledge to use Nadz Tools?

No. Nadz Tools is designed to be user-friendly with a simple interface. You can deploy tokens, set up locks, and manage vesting without any technical expertise.
