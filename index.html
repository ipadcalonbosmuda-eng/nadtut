Fitur Utama

Create Token

Buat token standar dengan satu klik. Dukungan metadata, initial supply, dan fee routing.

Token Locker

Kunci token untuk kepercayaan komunitas. Atur durasi & cliff, bukti on-chain transparan.

Liquidity Lock

Kunci LP untuk mencegah rug. Tampilan jadwal unlock & notifikasi.

Vesting

Pembagian token untuk tim & investor dengan skema fleksibel dan klaim mandiri.

Explorer Verify

Otomasi verifikasi kontrak di explorer Plasma (opsi alternatif bila Sourcify tidak tersedia).

Auditable

Jejak transaksi lengkap, export CSV, dan widget bukti untuk ditaruh di situs project.

Solusi untuk Builder

  • SDK ringan (JS/TS) untuk integrasi cepat
  • Komponen UI siap pakai (header top + sidebar di App)
  • Multi-account & multi-role (owner, team, auditor)
  • Siap PWA untuk App terpisah

Pertanyaan Umum

Kenapa situs utama dipisah dari Web App?

Untuk fokus: situs utama untuk informasi & SEO; App untuk interaksi on-chain (memerlukan wallet).

Apakah tema warna konsisten dengan Plasma?

Ya: latar putih/hitam dengan aksen hijau plasma. App akan membawa sidebar kiri & header atas (network, balance, wallet).

Apakah ada dukungan verifikasi kontrak otomatis?

Kami siapkan jalur alternatif saat Sourcify tidak tersedia, termasuk integrasi ke explorer Plasma.
