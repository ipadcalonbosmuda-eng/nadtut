Main Features

Create Token

Create a standard token with one click. Metadata, initial supply, and fee routing supported.

Token Locker

Lock tokens for community trust. Set duration & cliff, fully transparent on-chain proof.

Liquidity Lock

Lock LP to prevent rug. Unlock schedule & notifications included.

Vesting

Distribute tokens to team & investors with flexible schemes and self-claim options.

Explorer Verify

Automated contract verification on Plasma explorer (alternative if Sourcify is unavailable).

Auditable

Complete transaction logs, CSV export, and proof widgets to embed on project sites.

FAQ

Why separate the Main Site from the Web App?

For focus: the main site is for information & SEO; the App is for on-chain interaction (requires wallet).

Are the theme colors consistent with Plasma?

Yes: white/black base with plasma green accents. The App will carry the left sidebar & top header (network, balance, wallet).

Is automated contract verification supported?

We prepare alternative paths when Sourcify is unavailable, including Plasma explorer integration.
