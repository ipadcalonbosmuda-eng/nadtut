Features

Token Creation

Create tokens with configurable supply, name, symbol, and owner.

Token Locker

Lock tokens on-chain for community confidence. Choose duration and cliff periods, and generate public proof links for transparency.

Liquidity Locker

Protect LP positions by locking liquidity. Define unlock schedules and assure traders of long-term stability.

Vesting Manager

Set structured vesting schedules for teams, advisors, and investors. Supports cliffs, gradual release, and self-claim directly via the Web App.

Solutions for Builders

  • One-stop Web App: Token deployer, token lock, liquidity lock, and vesting in one interface.
  • Plasma-Only: No multi-chain modes — optimized purely for Plasma Network.
  • Transparent & Auditable: On-chain proofs and public explorer pages.
  • Security First: Verification and lock proofs integrated by default.

FAQ

Where do I deploy and manage tokens?

Inside the Plasmatic Web App at app.plasmatic.to. This site is for information only.

Which networks are supported?

Plasmatic is exclusive to Plasma Network. Cross-chain support is not planned.

Is contract verification automatic?

Yes. Contracts deployed in the Web App are prepared for verification on Plasma explorer.

Can I export reports for audits?

Yes. Export CSV reports and share public proof pages for locks and vesting schedules.
